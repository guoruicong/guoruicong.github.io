# Noah Portfolio

郭睿聪（Noah）的 UI / UX 与视觉设计作品集。

## 本地启动

```bash
npm install
npm run dev -- --port 8080
```

访问 `http://127.0.0.1:8080/index.html`。

## 页面

- `index.html`：作品集首页
- `projects.html`：项目索引与分类筛选
- `about.html`：个人介绍
- `case-meitu.html`：美图影像体验案例

## 技术

- React + Vite
- Motion
- GSAP
- React Bits 交互动效组件
